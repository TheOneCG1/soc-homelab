# Networking Verification

## Steps Performed
1. Verified IP addresses using `ip a` and `ipconfig`
2. Verified default gateway using `ip route`
3. Tested internet access with `ping 8.8.8.8`
4. Tested VM-to-VM connectivity
5. Established SSH connection to the SOC server

## Result
All networking tests were successful. The environment is ready for log ingestion and monitoring.
