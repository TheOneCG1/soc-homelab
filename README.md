This project documents the setup of a basic SOC homelab using a Mac and VMware Fusion. The goal of Day 1 is to build and verify the infrastructure and networking required for security monitoring.

## Environment
- Host: macOS (Intel Mac)
- Virtualization: VMware Fusion
- Virtual Machines:
  - Ubuntu Server (SOC server)
  - Windows (windows-endpoint)
  - Ubuntu Desktop (ubuntu-attacker)

## Day 1 Goals
- Create and configure virtual machines
- Verify internal networking
- Enable and test SSH access
- Document the environment

Day 2 will focus on SIEM installation and log ingestion.
